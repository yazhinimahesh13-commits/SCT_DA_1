# Excel Sales Dashboard – Superstore Dataset

This project is part of my Data Analyst Internship Task at SkillCraft Technology.

## Objective
Analyze the Superstore Sales dataset and build an interactive sales dashboard using Pivot Tables and Charts.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

## Dataset
Superstore Sales Dataset containing:
- Order Date
- Region
- Category
- Sub-Category
- Sales
- Profit

## Dashboard Features
The dashboard visualizes:

- Total Sales
- Profit by Category
- Sales Trend Over Time
- Top Performing Categories
- Low Performing Segments

## Key Insights
- Technology category generated the highest profit
- Some sub-categories show negative profit
- Sales increased during the last quarter
